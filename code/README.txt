1. KNN Classification:

python knn.py -demoMode -k -fileName

demoMode : "1" for Demo with filename as "", otherwise "0"

2. Naive Bayes:

python NaiveBayes.py -fileName

For running Demo data:

python NaiveBayesDemo.py -query

query example: sunny,cool,high,weak


3. Decision Tree:

python decisionTree.py -fileName


4. Random Forest:

python randomForest.py -fileName


5. Boosting:

python boosting.py -fileName
